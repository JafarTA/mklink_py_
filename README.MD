# Windows AppData 安全轉移與掃描工具 (Symlink Creator)

這是一個基於 Python 與 Tkinter 開發的圖形化工具，旨在解決 Windows 系統碟（C 槽）空間不足的問題。

## 🌟 功能特色

- 自動掃描：一鍵分析 `AppDataLocal` 與 `AppDataRoaming` 內佔用空間最大的資料夾。
- 安全轉移：採用「先複製、後改名備份」的邏輯，確保資料在轉移過程中不會因中斷而丟失。
- 符號連結 (Symlink)：自動執行 `mklink` 指令，讓軟體在路徑更改後仍能正常運作，不影響任何設定。
- 自動復原機制：若轉移或建立連結失敗，程式會自動將資料還原至初始狀態，保障系統穩定性。
- 管理員提權：自動檢查並請求 Windows 管理員權限，確保系統指令執行成功。

## 🚀 如何使用

1. 環境需求：
   - Windows 1011 系統
   - 已安裝 Python 3.x

2. 執行程式：
   - 以管理員身分執行 `mklink_auto_V3.py`。
   - 點擊「掃描 AppData 佔用空間」，找出佔用空間的軟體資料夾（如 Chrome, Spotify, LINE 等）。

3. 操作步驟：
   - 在清單中選擇欲轉移的項目（或手動輸入路徑）。
   - 選擇 D 槽或 E 槽的目標資料夾。
   - 點擊「開始安全轉移」，等待程式完成複製與連結建立。

## ⚠️ 注意事項

- 關閉軟體：在執行轉移前，請務必先關閉該資料夾對應的軟體（例如搬移 Chrome 暫存時請關閉 Chrome）。
- 磁碟格式：目標磁碟必須為 NTFS 格式，否則 Windows 不支援符號連結功能。
